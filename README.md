# News App (React + TypeScript)

A small **News Simulation App** with **Dark/Light theme** and **English/Arabic (EN/AR) localization**, built as part of the test project.

---

## Features
- Fetch posts (news) & authors using **RTK Query** (JSONPlaceholder API)
- **Dark/Light theme toggle** (animated + persisted in localStorage)
- **EN/AR language switch** with UI translations (persisted)
- **Skeleton loader** for loading states
- Thumbnail images with **fallback support**
- Navigation using **React Router**
- Responsive UI using **TailwindCSS**

---

## Tech Stack
React 18, TypeScript, Redux Toolkit (RTK), RTK Query, TailwindCSS, JSONPlaceholder API
